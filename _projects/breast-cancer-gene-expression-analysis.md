## Project Overview

This project explores breast cancer gene expression data to investigate how genetic and clinical variables relate to patient outcomes. The analysis focuses on identifying patterns within high-dimensional biological datasets and using data visualization techniques to better understand relationships between gene expression profiles and clinical characteristics.

Working with genomic datasets provided experience handling large biological datasets, performing data cleaning and preprocessing, and interpreting results within a biomedical context.

## Objectives

- Explore breast cancer gene expression datasets
- Investigate relationships between genetic and clinical variables
- Perform exploratory data analysis and visualization
- Identify patterns associated with patient outcomes
- Develop reproducible data analysis workflows

## Methods and Tools

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
